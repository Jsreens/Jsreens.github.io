Introduction to the issue

Displaying math equations

If you need to display a math equation in your presentation, Reveal uses a MathJax plugin to make this possible.

This allows the creator of the presentation to accurate display equations used using the correct characters and spacing. It eliminates the issue of improperly formatted equations when displaying findings.
